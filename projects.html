Computer Science Projects

Throughout my career as a Computer Science student, I have worked on several projects that showcase/exhibit my software engineering prospects and knowledge.

Many of these projects have since been archived, and all of their source code is publicly available on GitHub. Most contain bugs and unfinished aspects, though, they were never intended for publication/profit; my intention in creating them was to further my programming horizons.

Castlevania Legion

About The Game/Code

This project was made for my AP Computer Science Principles final. Thank you Andrew Matzureff.

Java was the programming language of choice. This project included nice particles, colors, and combat. Also, advanced and self-programmed physics, restitution, and other contact forces are implemented.

Swift (iOS) Calculator

About The App/Code

This calculator app was my first experience programming on Mac in XCode. This calculator has simple arithmetic operations.

Swift was the programming language. I implemented this app using a Queue data structure for the operators and operands. Square root, power, addition, subtraction, multiplication and division are included. Other features are planned in the future for complex operations.

Breakout

About The Game/Code

The AP Computer Science Principles Create Performance Task is to create an application that demonstrates knowledge in various skills, including abstraction, encapsulation, simplistic algorithm and functions, etc.

I decided to create a Breakout/BrickBreaker clone entirely from scratch in the Java programming language. Without the help of any major sources such as friends, teachers, etc. I attempted to integrate my game library/framework into it, which, at the time of making Breakout was very primitive in what it was. Nonetheless, I loved making this game, and it was a major step forward in what would further expand my gaming development horizons and the Java language.

Castlevania

About The Game/Code

Castlevania was the very first major game that I made in conjunction with my friend Brandon Willis.

Castlevania was a game constructed for my AP Computer Science A final project. The development process lasted over a month, and although the game is incredibly simplistic, it was the first major game I created in Java using Swing and AWT. In my AP Computer Science A class, we had a vote [amongst eighty students] to determine everyone’s favorite projects. Castlevania placed first in the 2015-2016 school year. At the time, I was a Junior.
