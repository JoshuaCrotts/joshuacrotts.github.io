Computer Science Projects

Computer Science Projects

Throughout my career as a Computer Science student, I have worked on several projects that showcase/exhibit my software engineering prospects and knowledge.

Many of these projects have since been archived, and all of their source code is publicly available on GitHub. Most contain bugs and unfinished aspects, though, they were never intended for publication/profit; my intention in creating them was to further my programming horizons.

Castlevania Legion

About The Game

This project was made for my AP Computer Science Principles final. We were to design a game or an application to our hearts content. My friend, Andrew Matzureff assisted with the physics and other miscellaneous operations. The sprites, images, and sound are all appropriately sourced in the code on GitHub. At the end, Castlevania Legion was entered into a contest with all the other games/applications created by other classmates (approximately 60 students) with some in pairs and groups of three. Castlevania Legion won first place out of all of these applications.

About The Code

Java was the programming language of choice. At first, when creating the physics, I was unsure of what to do concerning gravity and other contact forces. Andrew, however, programmed the engine with restitution, gravity, and other outstanding elements mimicking other more complex engines such as JBox2D, which we now use. It took us over a month to fully program the game. The state of it is merely a tech-demo at best, but I feel as for a final for a high school project (and for Andrew, a college project), it was technically and graphically advanced with nice particles, colors, and combat.
